# FinSphere AI Design System

Extracted from `stitch_finsphere_ai_super_app_ALL_FINAL/stitch_finsphere_ai_super_app/finsphere_ai/DESIGN.md`.

## Principles

- Dark-mode-first obsidian surfaces.
- Emerald primary actions and growth indicators.
- Glass panels with subtle white borders and blurred depth.
- Plus Jakarta Sans for headings, Inter for body/data, Geist for labels.
- Data-first layouts with tight scanning, responsive cards, and tabular financial figures.

## Components

- Buttons: emerald filled primary and glass ghost secondary.
- Forms: dark glass inputs with emerald focus glow.
- Cards: 16px radius, translucent surface container, 1px white border.
- Charts: Recharts with emerald/tertiary fills and dark-grid-friendly labels.
- Navigation: fixed desktop sidebar, compact mobile tab strip.
